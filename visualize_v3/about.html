About Econograph

Mapping the intellectual network of economists

What is this?

Econograph is an interactive visualization of the network of economists and their intellectual influences. By scraping data from Wikipedia and applying network analysis algorithms, we can see how economic ideas have flowed through history and identify the most influential thinkers in the field.

How it works

The project consists of three stages:

Stage 1: Data Collection

We start with Wikipedia's List of Economists, an alphabetically-organized index of notable figures. For each economist, we extract:

  • Basic information: Name, birth/death dates, photo
  • Intellectual influence: Who influenced their thinking
  • Academic lineage: Doctoral advisors and students
  • Unique identifier: Wikipedia page ID for reliable linking

The scraper is built with Scrapy and uses the wikipedia library to reliably resolve page IDs, even when names are ambiguous or redirects exist.

Stage 2: Network Analysis

Once we have the raw data, we construct a directed graph where:

  • Nodes represent individual economists
  • Edges represent influence relationships (A influenced B)

We then apply the PageRank algorithm (the same algorithm Google uses to rank web pages) to calculate a centrality score for each economist. Those with higher scores are more "influential" in the sense that they are referenced by many others, who themselves are well-connected.

Stage 3: Interactive Visualization

Finally, we visualize the network as a force-directed graph using D3.js. Key features include:

  • Node size reflects PageRank score (bigger = more influential)
  • Timeline axis positions nodes by birth year
  • Interactive exploration with search, click for details, drag to reposition
  • Real-time Wikipedia integration fetches biographies on demand

Technology Stack

Python Scrapy NetworkX D3.js Wikipedia API

Limitations

This project has some important limitations to keep in mind:

  • Data is limited to English Wikipedia and its editorial choices
  • Not all economists have complete infobox data (especially historical figures)
  • Influence relationships are often incomplete, subjective, or missing
  • The scraper uses heuristic parsing which may introduce errors

Despite these caveats, the dataset provides a fascinating lens into the development of economic thought and the scholars who shaped it.

Source Code

This project is open source. The code includes detailed comments and documentation, making it a useful learning resource for web scraping, network analysis, and data visualization.
